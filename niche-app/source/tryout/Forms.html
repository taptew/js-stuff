Basic form

All input, textarea, select elements with class .form-control are set width:100% by default. Wrap labels and groups in .form-group for optimal spacing.

Inline form

User .form-inline class for left aligned inline-block controls. This only applies to viewport that are >= 768px

Horizontal form

User .form-horizontal class along with .form-group make form behave like grid without needing .row class.

Other features

Check boxes, inline checkboxes and radio buttons.
